# Abstract
This project uses Machine Learning to predict shipment delivery status and segment customers based on their behavior. Supervised Learning algorithms are used for shipment prediction, while Unsupervised Learning algorithms are used for customer segmentation. The system helps improve logistics management, customer analysis, and business decision-making.

---

# Introduction
Logistics companies handle large amounts of shipment data daily. Predicting shipment delays and understanding customer behavior are important for improving services. This project aims to build an AI-based system for shipment prediction and customer segmentation using Machine Learning techniques.

---

# Literature Review
Many companies use AI and Machine Learning in logistics and supply chain management. Algorithms like Random Forest and K-Means are commonly used for prediction and customer analysis. Existing systems mainly focus on single tasks, while this project combines both shipment prediction and customer segmentation.

---

# Methodology
The dataset is collected from Kaggle and preprocessed using Python libraries. Supervised Learning models are trained for shipment prediction, and K-Means clustering is used for customer segmentation. The models are evaluated using accuracy scores and visualizations.
